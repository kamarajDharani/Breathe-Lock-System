# 🚗 BreatheLock – Smart Vehicle Safety System

An embedded safety system that prevents vehicle ignition when alcohol is detected in the driver’s breath.

---

## 🧠 Overview
BreatheLock is designed to reduce drunk driving accidents by automatically detecting alcohol levels and disabling engine ignition when unsafe conditions are detected.

---

## 🚀 Features
- 🍺 Alcohol detection using MQ-3 sensor  
- 🔒 Automatic engine lock using relay  
- 🔊 Buzzer alert system  
- 📟 LCD display for real-time status  
- ⚡ Low-cost embedded safety solution  

---

## 🛠️ Tech Stack
- Arduino UNO  
- Embedded C  
- MQ-3 Sensor  
- Relay Module  
- LCD Display  

---

## ⚙️ Working
1. Sensor detects alcohol in breath  
2. If above threshold:
   - Engine is stopped  
   - Buzzer alert is triggered  
3. Status shown on LCD
